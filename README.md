# 🏠 House Price Prediction App

This project is a Machine Learning web application built using Streamlit to predict house prices based on features like area, bedrooms, bathrooms, and garage capacity.

---

## 🚀 Features

- 📊 Real-world Kaggle dataset
- 📈 Data visualization (heatmap & scatter plot)
- 🤖 Linear Regression model
- 🔮 Interactive prediction UI
- 🌐 Web application using Streamlit

---

## 🛠️ Technologies Used

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Streamlit
